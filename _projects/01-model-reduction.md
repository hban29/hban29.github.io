This project addresses modeling, simulation, and analysis challenges for infinite-dimensional nonlinear systems governed by nonlinear partial differential equations.

Core contributions include nonlinear model-reduction methods based on geometric and data-driven principles, metric complexity measures, and closure strategies for stable reduced-order models.

Applications include aerodynamics, unsteady airflow in buildings, process systems, and high-dimensional spectral data analysis.
